Emlakjet House Price Prediction

Project Description

This project focuses on predicting real estate prices using machine learning techniques. We collected data from Emlakjet through web scraping, focusing on residential flats in Izmir and Istanbul. After preprocessing and feature engineering, multiple regression models were trained, including Ridge, Random Forest, Gradient Boosting, and an optimized LightGBM model. The optimized LightGBM achieved the best performance with an R² of 0.920 and MAPE of 9.78%. The study highlights the importance of district-level dynamics and market conditions in price prediction.
