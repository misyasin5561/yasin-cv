Anonmeet Project

Project Description

AnonMeet is a mobile application designed for Android that allows users to match anonymously based on shared hobbies and interests. Built with Flutter and powered by Firebase, the app provides a secure and minimalistic user experience focused on privacy and meaningful connections. Privacy First: Real names are only shared after matching no profile photos or detailed personal info up front. Interest-Based Matching: Smart algorithm connects users through common interests. Modern UX: Swipe-based interface, daily like limits, and a clean, user-friendly design. Firebase Backend: Real-time database, secure authentication, and cloud messaging. Rich Profiles: Users can highlight hobbies, music taste, favorite movies, sports, and more. Optimized for Android, AnonMeet offers a fresh take on social interaction anonymous, interest-driven, and secure.
